## 📚 Sobre o repositório

Este repositório foi criado com o objetivo de **praticar a linguagem C++ e manter a lógica de programação sempre afiada**.

Aqui serão adicionados exercícios, desafios e pequenos algoritmos desenvolvidos para treinar raciocínio lógico e melhorar a resolução de problemas utilizando C++.

## 🎯 Objetivo

- Manter a lógica de programação sempre em dia
- Praticar resolução de problemas
- Reforçar conceitos fundamentais de programação

## 🚀 Tecnologias utilizadas

- C++

## 🧠 Motivação

A prática constante é uma das melhores formas de evoluir na programação.  
Este repositório serve como um espaço para **treinar lógica, testar soluções e acompanhar a evolução no aprendizado **.
